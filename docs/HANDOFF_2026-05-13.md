### 2. Bar Chart Fill Parent Height

**Problem:** `ProgramRechartsBar` di `TransmisiSlide` overflow PanelCard (chartHeight formula N×rowHeight). Kalau N besar → X-axis "0% 25% 50% 75% 100%" ke-clip. Kalau rowHeight diperkecil → bar tipis "ga full".

**Fix:** Tambah prop `fillHeight: boolean` di `ProgramRechartsBar`. Bila true, `ChartContainer` pakai `height: "100%"` → Recharts ResponsiveContainer auto-distribute rows by parent. Margin bottom (28*fontScale px) reserved untuk axis labels otomatis aman.

**Cascading fix di parent containers** supaya flex height propagate:
- `SlideShared.PanelCard` body → tambah `display: flex; flexDirection: column`
- `TransmisiSlide` chart wrapper → `flex: 1, minHeight: 0, display: flex`
